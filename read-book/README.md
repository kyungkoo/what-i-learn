# 책 읽기

## 1. 기본 규칙
- 한 주에 한 장(chapter)씩 읽도록 한다.
- 읽은 장은 꼭 정리해서 github에 요약한다.
- 한 권을 다 읽으면 반드시 서평을 쓴다.
- 한번에 너무 많은 책을 읽지 않는다. (같은 분야에서 동시에 읽는 책은 최대 2권으로 제한)
- 책을 다 읽지 않은 상태에서 다른 책을 펴지 않는다.
